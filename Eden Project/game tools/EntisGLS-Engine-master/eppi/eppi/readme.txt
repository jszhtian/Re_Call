■ 初めに
　この度は「EPPI SDK」をダウンロードくださいましてありがとうございます。
　EPPI とは、e-Player Plugin Interface の略で、Entis Mutimedia Player（萌衣☆Player）のプラグインのインターフェースの仕様を取り決めたものです。


■ ファイル構成
　本アーカイブ内のファイル構成は以下のようになっています。

\ ルート
├─ include\
├─ epim_bmp\
├─ epex_bmp\
├─ eppi-spec.doc
└─ readme.txt （このファイル）

　EPPI 仕様書は、eppi-spec.doc です。MS-Word 形式のドキュメントファイルです。
　include フォルダには eppi.h が入っています。このファイルをインクルードして、EPPI インターフェースを実装することが出来ます。
　epim_bmp フォルダには BMP インポートプラグインのソースファイルが、epex_bmp フォルダには BMP エクスポートプラグインのソースファイルが入っています。


■ 著作権について
　本ソフトウェア（EPPI SDK）は無償にて配布していますが、著作権を放棄したわけではございません。
　著作権は、Leshade Entis、及び Entis-soft が所有しています。
　無断での再配布を禁止します。
　作成された EPPI プラグインは著作権的に本ソフトウェアは無関係ですので、作成者の意思で自由に配布することが出来ます。
　ヘッダファイルに関しては、プラグインのソースと添付する形であれば二次配布してもかまいません。
