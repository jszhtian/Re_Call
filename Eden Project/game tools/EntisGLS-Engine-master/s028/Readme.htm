No.028 CUFFS汎用 メッセージローダー
これはメーカー"CUFFS"のゲームシナリオを抽出するプログラムです。
メーカーと、このプログラムとは、なんら関係ないので、不具合が生じても
絶対メーカーに問い合わせをしてはいけません。

■利用条件■
　以下の条件に一つでも当てはまらない方は、このツールを使用できません。
　1.注意事項に全て目を通し、全て同意できること。
　2.解凍、ダブルクリック、コピーの方法が分かること。
　3.日本語を理解しコミュニケーションを取れること。
　4.ゲームをインストールしたフォルダが分かること。
　5.不具合が生じても自分で対処できること。

■利用方法■
　●ケース１『*.csxが見つかっている場合』（CUFFSとか）
　　1.シナリオファイルである「????.csx」をゲームをインストールしたフォルダより
　　　任意のフォルダ（以下、このフォルダを「フォルダＡ」といいます）にコピーしてください。
　　2.このツールを解凍し「cuffs_con.exe」と「uni2shi.tbl」と変換したいゲームの「setting.ini」を、
　　　「フォルダＡ」にコピーしてください。
　　3.「cuffs_con.exe」をダブルクリックしてください。
　　　（いつまでたっても終わらない場合は「Ctrl」+「C」で強制終了してください）
　　4.「cuffs_dec.txt」が「フォルダＡ」に作成されるはずです。
　　これが抽出されたテキストファイルです。

　●ケース２『*.csxが見つからない場合』（隣り妻とか）
　　「*.csx」が含まれると思われるnoaファイルを解凍します。（パスワードが必要な場合は普通の人には無理です）
　　　恵理ちゃんclub＞Utility＞乃亜アーカイバ
　　　次に「setting.ini」を例にならって書き換えてケース１に続きます。

　●ケース３『こんねこの場合』（ま〜まれぇど）
　　　説明を書くのに力尽きたので分からない人は使わないでください。
　　　修正パッチを当てる。「script.noa」を乃亜アーカイバで解凍。パスワードは「convini_cat」
　　　ケース１を実行後、「Repl-Ace」の置換計画にmodtxt.rplをセット、cuffs_dec.txtを書き換え対象にして一括置換
　　　おしまい

■注意■
・当たり前ですが、抽出されたテキストファイルは、メーカーにライセンスされており
　他人に配布することは禁止されています。配布した場合、メーカーならびに
　作者(nota)及び同様のツール製作者が極めて迷惑するので絶対行わないでください。

■アンインストール■
　EXEファイルを削除してください。
　このツールはレジストリを使用していません。

■動作環境■
　WindowsXP Home Editon SP2で動作を確認しています。
　他の環境での動作は確認しておりません。

■作成環境■
　このツールは「Borland C++ Compiler 5.5.1」でコンパイルされています。

■メモ■
　主人公の名前を変えたい人は「setting.ini」をいじってください。
　たていす氏の抽出ソフトActivePerl版（すでに消滅）と違う点
　・変換が早い（たていす氏の１時間に対し、10秒程度で変換可能）
　・Perlの実行環境が不要。
　・選択肢が表示される。
　・変換後Shift_jis(cp932?)になる。（たていす氏はUTF8で出力）
　・主人公の名前が変えられる。（無駄？）
　・発言者の名前取得可能に(ver1.1から)
　・setting.iniの編集で次回作も変換可能…になるかも？
　・そこはかとなくバグが潜んでいる気がする。
