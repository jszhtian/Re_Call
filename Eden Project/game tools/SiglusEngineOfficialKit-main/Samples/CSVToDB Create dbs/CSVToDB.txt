≪関連ツール≫
	CSV2DBS.exe
	データベースコンバートツールです。
	「RealLive開発環境\◇開発支援ツール\CSV2DBS_データベース作成ツール」の中に入っています。
	Excel で作成した csv ファイルを、dbs ファイルに変換します。

≪関連命令≫
DATABASEGET_NUM(DatabaseNo, ItemNo, ColumnNo, F[000])	データベースから整数データを取り出す
DATABASEGET_STR(DatabaseNo, ItemNo, ColumnNo, S[000])	データベースから文字列データを取り出す
DATABASEGET(DatabaseNo, ItemNo){(*), (*), …}			データベースから複数のデータを取り出す
Res = DATABASECHECK_ITEM(DatabaseNo, ItemNo)			データベースのアイテムを調べる（Res = 0: 存在しない、1:存在する）
Res = DATABASECHECK_COLUMN(DatabaseNo, ColumnNo)		データベースのコラムを調べる（Res = 0: 存在しない、1:整数フラグ、2:文字列フラグ）
Res = DATABASEFIND_NUM(DataBaseNo, ColumnNo, Num)		データベースから整数データを検索（Res = アイテム番号、見つからない場合は -1）
Res = DATABASEFIND_STR(DataBaseNo, ColumnNo, Str)		データベースから文字列データを検索（Res = アイテム番号、見つからない場合は -1）
Res = DATABASEFIND_STR_REAL(DataBaseNo, ColumnNo, Str)	データベースから文字列データを検索（大文字小文字を区別する）（Res = アイテム番号、見つからない場合は -1）

≪データベースの作り方≫
	データベースは、以下のステップで作成します。

	１．Excel を使って、データベースを作成する。
	２．csv 形式で保存する。
	３．CSV2DBS を使って、csv ファイルを dbs ファイルにコンバート
	４．Gameexe.ini に登録

≪ステップ１：データベースの作成≫
	データベースは Excel を使って作ります。
	（ﾃﾞｰﾀｻﾝﾌﾟﾙ\データベース\sample.xls 参照）
	以下の説明は、サンプルを見ながら読むと分かりやすいと思います。

	≪データベースの書き方≫
		「Ａ」の列には、コマンドまたはアイテム番号を書きます。
		コマンド「#DATANO」の行には、コラム番号を書きます。
		コマンド「#DATATYPE」の行には、コラムのデータタイプを書きます。
		詳細は、≪アイテム番号とコラム番号≫、≪データタイプ≫を参照してください。

	≪アイテム番号とコラム番号≫
		データベースには、アイテム番号とコラム番号が必要です。
		「Ａ」の列にある数がアイテム番号、コマンド「#DATANO」の行にある数がコラム番号です。
		スクリプトでは、これらの番号を指定することでデータを呼び出します。
		（例：「ともよ」を取り出すには、アイテム番号 = 4、コラム番号 = 1 を指定します。）
		番号は、0〜2147483647 の範囲の整数です。

		アイテム番号がない行や、コラム番号がない列のデータは、取り出すことができません。
		逆に言えば、それらの行や列には、何を書いても構いません。コメントなどを書き込んでください。
		サンプルの「★プレイヤー」「名前」などは、アイテム番号がないため、コメントとして扱われます。

	≪データタイプ≫
		データには、整数データと文字列データがあります。
		整数データは、RealLive の A,B,C,D,E,F,G,H,I,J,L,Z フラグに、
		文字列データは、RealLive の S,M,K フラグに読み込むことができます。

		コマンド「#DATATYPE」の行にある文字がデータタイプを表します。
		データタイプに V を指定したコラムのデータは整数データに、
		S を指定したコラムのデータは文字列データになります。

≪ステップ２：csv 形式で保存≫
	Excel で、「ファイル」→「名前をつけて保存」→
	ファイルの種類を「CSV (カンマ区切り) (*.cvs)」を選んで保存すると、
	csv 形式でファイルを保存することが出来ます。

≪ステップ３：コンバート≫
	CSV2DBS を使って、csv ファイルを dbs ファイルにコンバートします。

≪ステップ４：Gameexe.ini に登録≫
	Gameexe.ini の #DATABASE_FILENAME に、dbs ファイルを登録します。

	例：
		#DATABASE_FILENAME.000="database000.dbs"
		#DATABASE_FILENAME.001="database001.dbs"
