程序说明：
    本程序旨在提供一个age社作品通用的图片提取解决方案。
    程序用于从age社作品的.rio文件中，提取CG/背景/系统元素图片，输出为png格式。
    目前不支持マブラヴ以来的立绘/差分。
    今后如有进展，将不定期更新。
------------------------------------------------------------------------
使用说明：
    运行程序，选择资源文件和提取到的目录（默认为程序所在目录）。
    资源文件可以选择*.rio,*.rio.002,*.rio.003,*.rio.ruo1,*.rio.kdl等。
    如果选择*.rio，同名的.rio.002和.rio.003（如果存在）中的图片也将同时被提取。
    点击“Extract”按钮开始提取，log在右边显示。
    点击“Pause”按钮暂停/继续；点击“Stop”按钮中止。
    *注意：如果试图打开非资源文件或非age社作品的资源文件，存在程序崩溃的风险。
