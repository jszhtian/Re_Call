AlphaMovieリファレンス


コンストラクタ
AlphaMovie

メソッド
open ( メディアを開く )
clear ( デコード済みキューをクリアする )
showNextImage ( 次の画像を対象レイヤーへ描画する )
isPlaying ( 再生中かどうかを判定する )
play ( 再生開始 )
stop ( 再生停止 )
setPosition (レイヤー位置を指定する)
setNextMovieFile (次に再生するメディアを予約する)

プロパティ
numOfFrame ( 全フレーム数 )
frame ( 現在のフレーム番号 )
loop ( ループ再生有無 )
nextLoop ( 次に再生するメディアのループ再生有無 )
preloadSamples ( 先読みバッファ数 )
left ( 再生位置左端 )
top ( 再生位置上端 )
screenWidth ( 再生範囲の幅 )
screenHeight ( 再生範囲の高さ )
FPSScale ( 再生スケール )
FPSRate ( 再生レート )

AlphaMovie.preloadSamples
機能/意味
先読みバッファ数
タイプ
AlphaMovieクラスのプロパティ (読み書き可能)
説明
先読みバッファ数を表します。
バッファ数を多くすることで安定性を高めたり、少なくすることで応答性を高めることが出来ます。
通常は変更しません。
