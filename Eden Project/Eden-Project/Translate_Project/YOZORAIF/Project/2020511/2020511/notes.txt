extract_neat 里
是我重写了解包 打包等功能
(对我来讲代码干净了很多..当然也修了蛮多bug)
使用是 cmd cd到这个目录
或是在这个目录shift加右键打开powershell
然后 ruby xxx.rb 参数 

ruby extract.rb Rio1
把Rio1解包到 extract/Rio1 下面

ruby extract_ws2.rb Rio1
把 extract/Rio1 里的ws2中文本提取到 extract/Rio1/oritxts/
每个ws2对应一个txt 格式就是
文本
文本
角色名
文本
...

ruby pack_ws2.rb Rio1
把 extract/Rio1/reptxts 下的文本打包到 extract/Rio1/rep/
此步需要原ws2文件存在

ruby fix_ws2.rb
在打包为ws2后对某几个ws2进行修正
因为有些它里面是绝对寻址 尺寸不一会导致表现不正常 修修ok

ruby pack.rb Rio1
打包 Rio1 到当前目录 Rio1.arc
会检查 extract/Rio1/rep/ 下有没有同名文件 有则替换
