在改为格里高利历之前，日本曾经使用过一种被称作太阴太阳历的（或者说阴历）的历法。

//太陰太陽暦は、月の満ち欠けを１ヶ月として数え、太陽の運行をもって季節を調節していく暦法だった。
太阴太阳历是以月亮的盈亏交替为一个月，兼顾太阳的运行来调节节气的一种历法。

//新月、つまり月が完全に欠けて真っ暗になった日を{１日:ついたち}とし、そこから月が満ちて再び欠けて消えるまでを１ヶ月とする。
新月，也就是以月亮完全属于月亏的状态作为每月的1日，然后从这天开始到月亮完全变圆再完全变为月亏的状态为止算作一个月。

//今も細く欠けた月を「{三日月:みかづき}」と呼ぶが、あれは旧暦で言う\n「３日目の月」。

//「{十五夜:じゅうごや}」は１５日目に訪れる月で、満月。中秋の名月は、月がとくに綺麗に見える秋頃の十五夜（＝満月）を眺める日だったのである。
「十五夜」是指第十五天夜里的月亮，也就是满月。被称为中秋的名月也是能够特别清楚地看见月亮的秋天的第十五个夜晚。

//あれらの数字は、元々はちゃんと日付に対応していたのだ。

//だが、この数え方だと問題もある。

//月の満ち欠けは、およそ２９～３０日。それを一ヶ月とすると、１２ヶ月で３５４日にしかならない。１年が１１日ほど足りなくなってしまう。
