太阳终于落了下去。
ひかり
「呐、看！\n　外面已经暗下来了哦！」
把笔记本打开在那里假装学习的光、看着外面说道。
看着从家里带来的菜谱的沙夜、缓缓地抬起了头。
沙夜
「那、差不多了就开始把」
沙夜一边说着，一边打开了手提包、从里面将烟花取了出来。手提包里装着各种各样的东西。
暁斗
「抓在手里的烟花么」
就因为想玩这个、光那家伙、从傍晚开始就一直坐立不安的样子。
ひかり
「说道夏天的话肯定就会提到烟花不是么。\n　晓斗也来玩啊」
躺在床上看着从武一那里借来的漫画的我、也起身来到了院子里。
在蜡烛上把烟花点燃、将燃尽的烟花放在水桶里。
放烟花什么的还真是很久没玩过了啊。
ひかり
「呐呐沙夜、最开始玩哪个呢？\n　我准备玩这个ー」
光一开始就选了铁炮形状的烟花。
暁斗
「那个肯定要最后再放啊」
男生最喜欢的铁炮形状的烟花、一般都会保留到最后再玩，这是常识。
ひかり
「有什么嘛、我是从喜欢的食物开始吃的那种人啊」
暁斗
「你这旁门左道……」
铁炮形状的烟花被光抢先拿走了、让我优点懊恼。我也想玩那个啊。
沙夜
「那我就选这个把」
火药兹拉兹啦的燃烧着、各色各样的光芒在夜里闪耀着。
这个样子看上去很梦幻、但同时也很朴素。
沙夜
「真漂亮……」
ひかり
「快看快看、我这边可是很激烈哦！」
暁斗
「喂、不要拿着烟花到处转啊」
说道烟花的话、让我想起来以前烟花大会的时候。
前几天去的海水浴场、哈巴港附近的海边、今年应该也有烟花大会的。
从我家去那里实在太远、加上我也不是很喜欢拥挤的地方所以就一直没去过、但是烟花在夜空中绽放的样子很漂亮……
武一他、是这么告诉我的。
好像是和阵野一起去的。
虽然可能华丽程度有所不及、但是手拿式的烟花也很漂亮。
暁斗
「呵呵……」
沙夜
「怎么了、晓斗？」
暁斗
「没、想起来武一那小子以前在我面前显摆的事了」
ひかり
「啊ー、我也被他显摆过！\n　说是和阵野一起去哈巴港看烟花大会来着」
ひかり
「稍微有些不甘心啊……」
光、应该也很想去吧。\n和我们一起。
作为男朋友来说、只能说非常对不起了。
沙夜
「我的话、或许更喜欢现在这里的烟花呢」
ひかり
「为什么呢？」
沙夜
「因为能在很近的地方看着。\n　而且、可以只和喜欢的人一起玩」
现在这里、只有我和沙夜还有光。
ひかり
「唔ー……」
暁斗
「而且还说过光是等回来的巴士就等了一个小时哦」
ひかり
「额、这是真的么？」
电车里应该也非常拥挤吧。
ひかり
「那果然还是这里的烟花更好一点呢」
沙夜
「呵呵、这和你刚才说的可不一样哦」
是喜欢在远处看烟花然而会漏掉一些细小的地方、还是说、像这样在近距离感受着更好呢。
就像看星星更好呢、还是感受幸福的日常更好呢、和这个多少有些类似。
我们、一边感受着这小小的幸福、一边享受着这只属于我们的烟花。
沙夜
「接下来玩线香烟花吧」
ひかり
「我也要ー。\n　来比比谁的能燃烧的更久吧」
沙夜
「好啊」
暁斗
「我可不会手下留情哦」
ひかり
「一二三」
兹拉兹拉兹啦。
三根线香型烟花、绽放出了小小的火花。
这是烟花里、最平凡朴素的一种烟花了、但还是美的让人觉得虚幻。
我们三个、不知道什么时候这个火花就会熄灭、都在祈祷着它能够多燃烧一会。
ひかり
「……」
沙夜
「……」
被火光照亮的两人的表情、不知道为什么看上去和之前有些不一样、看上去有些悲伤。
ひかり
「……我现在、真的很幸福啊」
暁斗
「……？\n　那为什么会用这么悲伤的语气说出来呢」
ひかり
「这份幸福、就像夏天的烟花那样梦幻。\n　因为担心不知道什么时候就会像烟花那样突然消失不见了\n　所以有些不安」
沙夜
「……」
烟花、只能在很短的时间里绽放自己的美丽。这么说起来人也是一样的。
我们现在幸福生活着的每一天、会不会就像是只能在夏天绽放的烟花一般、燃尽之后就会消失呢。
然后、正因为知道终有一天会结束、会不会感受到更深层次的幸福呢。
事实上、像这个夏天一样的日子、并不能长久的持续下去。
因为我能在这个家里生活的时间也仅仅是这个夏天而已。
沙夜
「……」
沙夜悲伤的凝视着火光渐渐微弱的线香烟花。
请不要熄灭――这样的愿望也是虚幻的、最终、火光还是熄灭了然后掉了下来。
光的线香型烟花也几乎在同一时间熄灭之后掉了下来。
暁斗
「看来是我赢了啊」
我手里的烟花、还有一点点的火光在跳动着。
在两人的凝视中、火光也渐渐收束、最终消失了。
然而……。
ひかり
「阿勒？」
沙夜
「……没有掉下来呢」
火光熄灭之后、就这样黏在上面并没有掉下来。
ひかり
「它好像在说、怎么可能就这么轻易掉下来啊」
沙夜
「呵呵、真像晓斗呢」
暁斗
「你们啊、不要一下子就消沉下去啊。\n　夏天还在继续、如果火光熄灭了的话、\n　ま再一次点燃不就好了么」
我给了两个人新的烟花。
ひかり
「是啊」
沙夜
「下次要做到比这坚持更久」
两个人重拾了一下心情、再一次享受起了一起放烟花的乐趣。
一边笑着、一边变得开朗起来。
