沙夜

沙夜
「可是、能把这些话都和人倾诉出来、\n　好歹心里好受了些」
沙夜
「三个人一起交往……吗」
沙夜
「……要是真的能做到的话、\n　那肯定也是非常幸福呢」
要是和其他人还不好说、但是和光的话我并不讨厌。
沙夜
「……我觉得。大概。一定会」
说着、回想起正巧撞见光和晓君接吻时我那混乱心情、又不禁觉得果然三个人还是太勉强了。
沙夜
「而且啊、晓君之所以会那样说、\n　肯定是因为不喜欢人家了啦」
沙夜
「实际上只喜欢光罢了、\n　却又这么温柔地在照顾我的情绪」
沙夜
「……只是这份温柔、反倒叫人格外难受啊」
小声地自言自语着、心情又低落了起来。
就在这时、我发觉自行车的声音从背后传来，由远及近。
这是我再熟悉不过的声音。
沙夜
「是晓君！？」
我头也没回、慌忙找地方想要躲起来。
骑着自行车行驶在去往沙夜家的路上、我半路捏了刹车。
一路骑得很急、自行车停下后、我先调整了一下呼吸。
而后、我就朝电线杆说起话来。
晓斗
「沙夜」
电线杆
「……」
电线杆没有理睬我。
不、我这话显然是和躲在电线杆后面的人说的。
晓斗
「我可知道你就在那里的。\n　我都看见了」
或许是听到了我自行车的声音吧、沙夜慌不择路地躲进了电线杆的阴影里。
然而、这条笔直大道也没有其他遮挡物、沙夜的身影是一目了然。
沙夜
「呜呜……只是觉得太难为情了」
沙夜放弃挣扎走了出来。
沙夜
「有什么事吗……？」
晓斗
「把头抬起来吧」
沙夜
「不要」
沙夜的小脑袋低得不能再低、铁了心不看我。
沙夜
「……我必须要回家帮妈妈做家务了」
晓斗
「那至少让我好好看看你嘛」
沙夜作势要逃，我不由分说地拦在她面前说道。
晓斗
「这都多少天没见着你了。\n　我可是很寂寞哦」
沙夜
「……」
晓斗
「沙夜」
沙夜
「要是见到晓斗君的脸了、我肯定会突然暴怒哦。\n　没准会像光一样、给你一拳头呢」
晓斗
「是生我的气吗？」
沙夜
「这还用说嘛。谁让你说那种话的……」
沙夜
「太差劲了。\n　我已经再也不想看到晓斗君的脸了」
晓斗
「沙夜……」
沙夜在心灵上拼命地想要疏远我，把我推开。仅仅是知道了她的心情就让我很难受。
我也不知道该说些什么话好了。
无论说什么、在沙夜听来都是自圆其说自私自利的谎言吧。
晓斗
「……第一颗星星已经亮起来了。\n　是长庚星吧」
沙夜
「？」
沙夜毫无防备地抬起头想要看星星、我立刻吻了上去。
沙夜
「――――」
双唇相接的瞬间、沙夜整个人都呆住了僵在那里。
我不管不顾地继续加深这个吻、并温柔地握住了她的手。
沙夜
「…………」
沙夜想是想要说些什么、却因为我堵住了她的唇说不出半个字。
品味了一会儿这个吻、我稍稍离开她的唇。
沙夜
「……这也太狡猾了」
晓斗
「我也想让沙夜明白我的真心啊。\n　出其不意的偷袭吻、这下就扯平了吧」
晓斗
「我喜欢你。沙夜」
沙夜
「晓君现在做的事、就是最差劲的耍弄女孩心意……」
说着、沙夜却闭上了眼。
而后、主动地吻了过来。
沙夜
「嗯唔……」
夕阳笼罩在重叠的两人身上、拉出了长长的影子。
我们已经彼此互通了心意、却没有结束这个吻的打算。
为了让对方知道几日不见的落寞、和一直以来擦身而过的心酸。
晓斗
「……我们是两情相悦呢」
沙夜
「嗯……。我喜欢你、晓君」
沙夜轻轻嗫嚅着、声音甜蜜得快要融化掉。
