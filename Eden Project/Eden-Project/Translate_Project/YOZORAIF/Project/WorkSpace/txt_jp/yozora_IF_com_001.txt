７月７日、七夕。
織姫と彦星が、年に一度の{逢瀬:おうせ}を楽しみにしているその日は、しかし、例年のように曇り空だった。
さをとめは大手コンビニチェーンと違って店内放送がないので、お客さんがこないと店内はシーンと静まり返ってしまう。
そんな静寂の中、紙をなぞるシャーペンのカリカリという音が微かに響いている。
沙夜とひかりが喫茶スペースで、俺は店番をしながらレジカウンターで、それぞれ勉強をしているせいだ。
ただし、ひかりはもうだれ気味だった。
ぐでーんとカウンターに突っ伏して、真面目に勉強をしている沙夜の横顔を、何故かちょっと恨めしそうに見ている。
ひかり
「ねぇ、沙夜～。\n　沙夜ってばぁ～」
沙夜
「なぁに」
ひかり
「そろそろ休憩しようよぉ」
沙夜
「もうちょっとしたらね」
ひかり
「……素っ気ない。そんなに勉強が好きか。\n　貴重な青春の１ページを、こんな事に費やして\n　いいのだろうか」
暁斗
「今の１ページのために、将来の何十年かが\n　台無しになっても知らないぞ」
ひかり
「お母さんみたいな事言うのやめてっ」
家でも相当言われているらしい。
３年生の俺たちは今年で卒業。その後の進路は様々だが、とりあえず三人とも大学へ行くと決めている。
今やっているのは、いわゆる受験勉強だった。
ひかり
「お団子食べちゃおっかなぁ」
沙夜
「ダメよ。それは七夕用のやつなんだから」
ふたりの間に団子が置いてあるのだが、さっきうちの店で買った物だ。
ひかり
「……お団子って七夕じゃなくて十五夜だよね」
十五夜、または{中秋:ちゅうしゅう}の名月。９月中旬から１０月上旬の満月の夜に、お月見を楽しむ行事。
月は暦と関係しており、古くは信仰の対象だった。お月様に団子などを供えて収穫のお祝いをするのが十五夜なのだ。
つまり、七夕とは一切関係ない。
夜空を見上げてわいわいやるイベントだからと、店長が間違って団子を仕入れたのだが、関係ないし曇ってるしで、早くも半額シールの貼られてたのを沙夜が一つ買ってくれた。
ひかり
「今夜は雨だから、どうせ七夕なんかやれないよ。\n　いいでしょ、食べちゃって」
沙夜
「……」
ひかり
「ねぇねぇ、沙夜ぁ～」
袖を引っ張って気を引こうとするが、集中している沙夜はまるで相手にしない。
本当にすごい集中力だ。尊敬する。
諦めたひかりは、独り言をボヤきだす。
ひかり
「……あーあ、結局３年間彼氏できなかったなぁ」
ひかり
「あたしさぁ、大きくなったら自然と彼氏とか\n　できるもんだって思ってた」
ひかり
「あと１０年くらいしたら結婚とかしてるつもりで\n　いるんだけど、もしかして楽観的すぎるかしら……」
沙夜
「そのためにも勉強をちゃんとしておくんだよ、ひかり。\n　何かあっても一人で生きられるようにね」
ひかり
「そんな後ろ向きな理由で受験勉強したくない！」
それについてはひかりに同意したいところだが、身よりのない俺にとっては切実な理由だった。
大学へ行けば、ここの下宿も出る事になるだろうし、その先となるとどこで何をしているか想像もできない。
ひかり
「……このまま、キスもしないで卒業してっちゃうのかなぁ」
ノートの端っこに落書きしながら、ひかりが悲しげに言う。
ひかり
「沙夜はいいよね…………暁斗とキスしたし」
沙夜
「っっ――――」
沙夜
「なっ、ななな、なにバカな事言ってるのよバカぁっ」
ひかり
「バカって２回も言った！？」
沙夜
「ぅぅ……」
沙夜が気まずそうに俺を見る。
暁斗
「そんな事もあったなぁ」
俺も動揺していたが顔には出さず、なんて事ない調子で返した。
ひかり
「あたしもしたいなぁ～。ねぇ暁斗ぉ、\n　キスしよっか」
暁斗
「バカかおまえはっ！」
いかん、ひかりがバカすぎて結局取り乱してしまった……。
ひかり
「えー、いいじゃーん。減るもんじゃなしー。\n　ねぇ、沙夜」
沙夜
「……知らない」
ひかり
「？　なに拗ねてるのよー」
沙夜
「バカひかり」
ひかり
「またバカって言われた！？」
沙夜はツンとした顔で、ひかりを無視する事に決めたみたいだった。
ひかり
「あたしは想い出がほしいだけなのにぃ……」
ひかりは一人で、ぐじぐじといじけてしまった。
店長
「おまたせ{宙見:そらみ}君、もういいよ」
奥から出てきた店長に言われ、俺はエプロンを外す。
店長
「これ鍵ね。雨降る前に下見行っといで」
暁斗
「ありがとうございます店長」
ひかり
「暁斗、どっか行くの？」
暁斗
「こないだ言ってた新しいアルバイトだよ。\n　今から下見に行くんだ」
ひかり
「あーあれか！　ついてっていい？」
暁斗
「いいけど……見に行くだけだぞ？」
沙夜
「ひかり、勉強まだ途中でしょ？」
さっきの事もあって、沙夜が厳しく言う。
店長
「机もエアコンもあるから、あっちで勉強しておいで」
ひかり
「だって！」
沙夜
「……しょうがないなぁ」
ひかり
「なによー。じゃあ沙夜だけここで勉強してれば？」
沙夜
「わたしは別に、いつでも見に行けるから\n　それでもいいけど」
そうは言いつつ、沙夜も勉強道具を片付けている。
暁斗
「それじゃ、三人で行くか」
