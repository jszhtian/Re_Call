pee ( Particle Effect Editor)

パーティクルエフェクトを作るためのツールです。
画像は PNG のみ使用可能です。

64bit環境では、pee64.exe を、32bit環境では、pee32.exe を起動してください。


制限事項
ライセンスキーを購入して登録していない場合、ファイルの読込みと保存、レンダリングは使用できません。

アンインストール
アンインストールするには uninstall.reg をダブルクリックします。
