//

#SAMPLE_BASIC


//----------------------------------------------------------
//最初は幕を降ろしておく
//----------------------------------------------------------
\FOUT(0)


//----------------------------------------------------------
//ＢＧＭ再生
//----------------------------------------------------------
\BGM(bgm03)

ＢＧＭを再生しました。


//----------------------------------------------------------
//ＳＥ再生 (単発)
//----------------------------------------------------------
\SE(d_chime01) //チャイム

単発ＳＥを再生しました。


//----------------------------------------------------------
//ＳＥ再生 (ループ再生)
//----------------------------------------------------------
\SE(semi,,,0) //セミ

ループＳＥを再生しました。


//----------------------------------------------------------
//ＳＥ停止
//----------------------------------------------------------
\SE(,1000) //ファイル名書かないと停止

ループＳＥを１秒かけて停止しました。



//----------------------------------------------------------
//カメラをXYZ=(0,0,0)に配置した状態で、
//背景 bg_030 を表示する
//----------------------------------------------------------
\BG.CMXYZ(0, 0, 0)
\BG(bg_030, , 0, 0)

背景を読み込みました。ただし、まだ最前面に幕が降りているので、何も見えません。



//----------------------------------------------------------
//幕をあげる(フェードインする)
//----------------------------------------------------------
\FIN(1000)

最前面の幕を上げました（フェードインしました）。これで後ろにあるスプライトが初めて見えるようになります。


//----------------------------------------------------------
//立ち絵の新規表示
//
//  優理立ち絵を500msかけてクロスフェード(新規表示)
//
//----------------------------------------------------------
\優理(YUU_1A01, 500)

立ち絵を表示します。


//----------------------------------------------------------
//立ち絵の移動
//
//  Xを+100ピクセル分動かしつつ
//  優理立ち絵を500msかけて(同じ画像に)クロスフェード(画像切替)
//
//----------------------------------------------------------
\SP.SX(100, 2000)
\優理

立ち絵を画像はそのままで、移動してみます。


//----------------------------------------------------------
//立ち絵移動しながら画像切り替え
//
//  Xを-100ピクセル分動かしつつ
//  優理立ち絵を500msかけてクロスフェード(画像切替)
//
//----------------------------------------------------------
\SP.SX(-100, 2000)
\優理(YUU_1A03, 500)

立ち絵を画像を切り替えつつ、移動してみます。


//----------------------------------------------------------
//立ち絵の消去
//
//  優理立ち絵を500msかけてクロスフェード(画像消去)
//
//----------------------------------------------------------
\優理(, 500) //ファイル名かかずに時間だけ指定すると消去になる

立ち絵を消去してみます。


//----------------------------------------------------------
//幕をさげる(フェードアウトする)
//  その際、ただのフェードではなく、
//  トランジション画像５番を使用してトランジションしてみる
//----------------------------------------------------------
\FOUT(1500, 5)

幕を下ろしました（フェードアウトしました）。


//----------------------------------------------------------
//ＢＧＭを停止しました。
//----------------------------------------------------------
\BGM(,1000) //ファイル名を書かないことで停止になる

ＢＧＭを停止しました。


//----------------------------------------------------------
//１秒待つ
//----------------------------------------------------------
\WA(1000)

１秒待ちました。



サンプルは以上です。



//----------------------------------------------------------
//幕を上げる(次のために幕を上げておかないと見えないまま)
//----------------------------------------------------------
\FIN(0)

//----------------------------------------------------------
//選択肢へ戻る
//----------------------------------------------------------
\GO(基本的な命令のサンプル)
