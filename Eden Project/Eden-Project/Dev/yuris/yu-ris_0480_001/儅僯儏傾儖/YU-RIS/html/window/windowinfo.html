■ウィンドウ情報取得命令 WINDOWINFO[ ]


ウィンドウに関する各種情報を取得します。


キーワード



NO = 値(0〜3)対象のウィンドウ番号を指定します
※０がメインウィンドウ、１〜３がサブウィンドウになります


LET = 変数名値を格納する変数を指定します


EXIST = 値(0/1)ウィンドウが存在するか否かを調べる場合は１を指定


E = 値(0/1)ウィンドウの有効／無効を調べる場合は１を指定


ACTIVE = 値(0/1)ウィンドウがアクティブになっているか否かを調べる場合は１を指定


CAPTION = 値(0/1)ウィンドウのキャプションを調べる場合は１を指定


X = 値(0/1)ウィンドウのＸ座標位置を調べる場合は１を指定


Y = 値(0/1)ウィンドウのＹ座標位置を調べる場合は１を指定


SX = 値(0/1)ウィンドウサイズＸを調べる場合は１を指定


SY = 値(0/1)ウィンドウサイズＹを調べる場合は１を指定


FULLSCREEN = 値(0/1)フルスクリーン(=1)／通常スクリーン(=0)を調べる場合は１を指定


FULLSCREENBUTTON = 値(0/1)最大化ボタンの有効(=1)／無効(=0)を調べる場合は１を指定


MINIMIZE = 値(0/1)ウィンドウが最小化しているか(=1)否か(=0)を調べる場合は１を指定


CLOSEMODE = 値(0/1)閉じるボタンを押したときの挙動を調べる場合は１を指定






[1]サブウィンドウ１が存在しているか調べる。

　　WINDOWINFO[NO=1 EXIST=1 LET=@CHECK]


[2]メインウィンドウがフルスクリーン状態か否か調べる。

　　WINDOWINFO[NO=1 FULLSCREEN=1 LET=@CHECK]


[3]メインウィンドウが最小化状態か否か調べる。

　　WINDOWINFO[NO=1 MINIMIZE=1 LET=@CHECK]
