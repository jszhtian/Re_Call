■TOP＞命令一覧＞レジストリ命令＞REGINFO［］
　REGINFO［］ 

■レジストリ情報取得命令 REGINFO[ ]


レジストリより下の階層にある任意のキーからデータを取得します。
キー名はフルパスで指定します。
※ver0.235 より、HKEY_CURRENT_USER だけでなく、他のキーの情報も取得できるようになりました。


キーワード



KEY = キー名キー名を指定します。


NAME = 名前名前を指定します。


LET = 変数名各種情報値を格納する変数を指定






[1]「HKEY_CURRENT_USER/Software/Firstia/ABC-Game」というキーの中の「ver」の名前に対応するDWORD値を取得する。

　　REGINFO[KEY="HKEY_CURRENT_USER/Software/Firstia/ABC-Game" NAME="ver" LET=@A]

[2]「HKEY_CURRENT_USER/Software/Firstia/ABC-Game」というキーの中の「dir」の名前に対応する文字列を取得する。

　　REGINFO[KEY="HKEY_CURRENT_USER/Software/Firstia/ABC-Game" NAME="dir" LET=$A]
