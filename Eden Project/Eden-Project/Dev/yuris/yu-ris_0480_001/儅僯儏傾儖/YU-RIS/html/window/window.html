■ウィンドウ設定命令 WINDOW[ ]


サブウィンドウを生成したり、位置やキャプションを設定したり、また「閉じる」ボタンを押したときの挙動を制御します。


キーワード



NO = 値(0〜3)対象のウィンドウ番号を指定します
※０がメインウィンドウ、１〜３がサブウィンドウになります


CAPTION = 文字列ウィンドウのキャプションを設定


A = 値(0〜256)ウィンドウ表示(=1〜256)／非表示(=0)を設定
※本来、A キーワードはウィンドウの不透明度を設定するための
機能なですが、正式実装はβ５からになり、β４の段階では
表示／非表示としてしか機能を持ちません。


E = 値(0/1)ウィンドウ表示(=1)／非表示(=0)を設定


X = 値ウィンドウのＸ座標位置を設定


Y = 値ウィンドウのＹ座標位置を設定


Z = 値(0〜3)ウィンドウの重なり順序を設定
0=前面(通常)／1=常に手前に表示／2=最背面に表示／
3=メインウィンドウより後ろに表示


SX = 値ウィンドウサイズＸを設定


SY = 値ウィンドウサイズＹを設定


ACTIVE = 値(0/1)ウィンドウをアクティブにする場合に１を設定


FULLSCREEN = 値(0/1)フルスクリーン(=1)／通常スクリーン(=0)を切り替えます


FULLSCREENBUTTON = 値(0/1)最大化ボタンの有効(=1)／無効(=0)を設定


MINIMIZE = 値(0/1)ウィンドウの最小化(=1)／通常化(=0)を切り替えます


CLOSEMODE = 値(0〜2)閉じるボタンを押したときの挙動を設定


CGID = 文字列表示対象のレイヤセットを指定します


MENUID = 文字列メニューバーとして設定するレイヤセットを指定します


REGIONID = 文字列ウィンドウの形を設定する画像があるＣＧレイヤを指定します


DRAGID = 文字列ドラッグできる場所を設定する画像があるＣＧレイヤを指定します






[1]サブウィンドウ１を生成します。
　(生成にはCAPTIONキーワードの指定が必須です)

　　WINDOW[NO=1 CAPTION="テストウィンドウ" SX=320 SY=240]


[2]メインウィンドウを非表示にします。

　　WINDOW[NO=0 E=0]


[3]メインウィンドウを常に手前に表示するようにします。
　 ※常に手前に表示を解除する際は 1 以外の値を指定することで解除になります。

　　WINDOW[NO=0 Z=1]


[4]サブウィンドウ２をアクティブにします。
　　※但し、逆に ACTIVE=0 としても、非アクテイブには出来ません。

　　WINDOW[NO=2 ACTIVE=1]


[5]メインウィンドウをフルスクリーン表示します。
　※ MOVIE 命令にてムービー再生中はスクリーン切り替え出来ません。

　　WINDOW[NO=0 FULLSCREEN=1]


[6]サブウィンドウ１に表示させるレイヤセットを「ABC」に設定します。

　　WINDOW[NO=1 CGID=ABC]


[7]メインウィンドウに表示させるメニューバーを、
　　メニューレイヤセットのルートフォルダの「M0」に設定します。

　　※システム起動時、メニューレイヤセットのルートフォルダはデフォルトで「M0」になり、
　　　その階層の下にレイヤを生成していく形になっています。

　　WINDOW[NO=0 MENUID=":/M0"]


[8]メインウィンドウの形を、ＣＧレイヤ「MASKCG」の画像に合わせて設定します。

　　※レイヤ「MASKCG」の画像の黒い色(0x000000)の部分が、ウィンドウの透過される部分になります。
　　　それ以外の色は無視されます。

　　WINDOW[NO=0 REGIONID="MASKCG"]


[9]メインウィンドウのドラッグできる部分を、ＣＧレイヤ「MASKCG」の画像に合わせて設定します。

　　※レイヤ「MASKCG」の画像の黒い色(0x000000)の部分が、ドラッグ「できない」部分になります。
　　　それ以外の色の部分はドラッグ「できる」部分になります。

　　WINDOW[NO=0 DRAGID="MASKCG"]
