■メニューレイヤ表示命令 MENUACT[ ]


メニューレイヤセットをコンテキストメニュー(右クリックメニュー)として表示します。


キーワード



ID = 文字列レイヤセットＩＤを指定


CONTEXT = 値(0/1)コンテキストメニューを表示する場合は１を指定






[1] メニューレイヤセット「ITEM」内のメニューレイヤをコンテキストメニュー(右クリックメニュー)として表示する。

　　MENUACT[ID="ITEM/" CONTEXT=1]
