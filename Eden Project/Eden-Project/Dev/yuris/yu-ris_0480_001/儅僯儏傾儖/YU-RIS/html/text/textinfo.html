　TEXT［］ 　TEXTINFO［］ 
■テキスト情報取得命令 TEXTINFO[ ]


テキストに関する各種情報を取得します。


キーワード



LET = 変数名情報値を格納する変数を指定


CX = 値(0/1)テキストカーソル座標Ｘを調べる場合は１を指定


CY = 値(0/1)テキストカーソル座標Ｙを調べる場合は１を指定


CLICK = 値(0/1)現在クリック待ち状態か否か調べる場合は１を指定


AUTORETURN = 値(0/1)自動改行設定か否か調べる場合は１を指定


AUTOPAGE = 値(0/1)自動改頁設定か否か調べる場合は１を指定


AUTOCLICK = 値(0/1)自動クリック待ち設定か否か調べる場合は１を指定


OUTPUTMODE = 値(0/1)テキスト出力モードを調べる場合は１を指定






[1] テキストの出力モードを調べる。

　　TEXTINFO[LET=@CHECK OUTPUTMODE=1]


[2] クリック待ち状態か否かを調べる。(基本的に別タスクから使用)

　　TEXTINFO[LET=@CHECK CLICK=1]
