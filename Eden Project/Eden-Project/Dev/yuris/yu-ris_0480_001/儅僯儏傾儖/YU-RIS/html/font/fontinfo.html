■TOP＞命令一覧＞フォント系命令＞FONTINFO［］
　FONT［］ 　FONTINFO［］ 
■フォント情報取得命令 FONTINFO[ ]


テキストフォントに関する各種情報を取得します。


キーワード



LET = 変数名情報値を格納する変数を指定


NO = 値NAME でフォント番号を指定する時に使用


COLOR = 値(0/1)テキストの現在の色を調べる場合は１を指定


COLOR2 = 値(0/1)テキストの現在の色を調べる場合は１を指定


SX = 値(0/1)テキストの現在の大きさ(横幅)を調べる場合は１を指定


SY = 値(0/1)テキストの現在の大きさ(縦幅)を調べる場合は１を指定


BOLD = 値(0/1)テキストが現在太字に設定されているか否かを調べる場合は１を指定


ANTIALIASING = 値(0/1)テキストのアンチエイリアスを調べる場合は１を指定


DRAWMODE = 値(0/1)テキストの現在の描画方法を調べる場合は１を指定


NUM = 値(0/1)テキストフォントの数を調べる場合は１を指定


NAME = 値(0/1)テキストフォントの名前を調べる場合は１を指定






[1] 現在設定されているテキストサイズ(横幅)を調べる。

　　FONTINFO[LET=@CHECK SX=1]


[2] 現在インストールされているフォントの総数を調べる。

　　FONTINFO[LET=@CHECK NUM=1]


[3] 現在設定されているフォントの名前を調べる。

　　FONTINFO[LET=$NAME NAME=1]


[4] 現在インストールされているフォントリストの３番目の名前を調べる。

　　FONTINFO[LET=$NAME NAME=1 NO=3]
