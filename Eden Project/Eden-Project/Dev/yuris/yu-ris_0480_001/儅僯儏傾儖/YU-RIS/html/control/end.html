■終了命令 END[ ]


ユーリスの実行を終了します。
RESTART キーワードに1を指定した場合は、ユーリスを再起動します。


キーワード



RESTART = 値(0/1)再起動する場合は1を指定





[1] ユーリスの実行を終了します。

　　END[]


[2] ユーリスの実行を一旦終了し、その後ユーリスを再起動する。

　　END[RESTART=1]
