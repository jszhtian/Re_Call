■無条件ジャンプ命令 GO[ ]


指定したラベルの位置に無条件で移動します。


キーワード



# = ラベル名ジャンプ先のラベルを指定




例：　　GO[#="ABC"]


ファイル内ジャンプや、別ファイルへのジャンプが出来ます。
別のファイルへジャンプする場合に、ジャンプ先のファイル名(右下図の例ではBBB.YST)を
記述する必要はありません。ラベル名の記述だけでＯＫです。

ただこの仕様も一長一短で、ラベルの書かれているファイル名をいちいち気にしなくて
いいというメリットがあるのですが、デメリットとしては、ファイル毎に同名のラベルを
作ることが出来ないという点です。ラベルが増えていったときに、うっかりラベル名が
重複してしまうという事態も考えられます。

それに対処するため、今後のバージョンアップにおいて、「重複しているラベルを指定する
場合」にのみファイル名の記述を強制するように変更する予定です。
