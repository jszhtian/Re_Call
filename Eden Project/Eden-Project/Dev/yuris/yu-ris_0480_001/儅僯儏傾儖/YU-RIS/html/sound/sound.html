■サウンドレイヤ設定命令 SOUND[ ]


ＢＧＭや効果音（ＳＥ）をサウンドレイヤに読み込み、再生します。
OGGファイル または WAVEファイルが指定できます。
(※特殊コーデック形式のWAVEファイルには対応していません)

ファイルの拡張子は記述を省略することができます。その場合、
拡張子を探しに行く順番としては
「(拡張子なし)」「.OGG」「.WAV」の順になります。

例えば、FILE="bgm01" と記述した場合、
まず"bgm01"という拡張子無しのファイルを探しに行きます。
なければ次に"bgm01.ogg"が有るかどうかを探しに行き、
それでもなければ"bgm01.wav"を探しに行きます。


キーワード



ID = 文字列サウンドデータを置くレイヤＩＤを指定


FILE = ファイル名再生するサウンドファイルを指定


E = 値(0/1)サウンドレイヤの有効無効を指定


PLAY = 値(0/1)１で再生、０で停止します


LOOP = 値０＝無限ループ、１以上＝再生回数


FADE = 値フェードイン／アウトの時間を設定します


PAUSE = 値(0/1)１で一時停止し、０で再開します。


START = 値再生開始位置を指定(ミリ秒単位)
マイナス値で遅延再生になります。


RESTART = 値ループして２度目以降に再生される際の再生開始位置を指定(ミリ秒単位)
マイナス値で遅延再生になります。


END = 値終端位置／ループ位置を指定(ミリ秒単位)


VOLUME = 値(0〜256)音量を設定します


PAN = 値(0〜512)パン(左右の位置)を0〜512で設定します(左〜右)　※デフォルトは256


PITCH = 値(0〜512)ピッチ(再生速度)を0〜512で設定します(0.5倍〜1.5倍)  ※デフォルトは256






[1]「ABC.ogg」ファイルをサウンドレイヤ「BGM」にロードし、ループ再生する。

　　SOUND[ID=BGM FILE="ABC.ogg" LOOP=0 PLAY=1]


[2]「ABC.ogg」ファイルをサウンドレイヤ「BGM」にロードし、１．５秒かけてフェードインしながら、３回再生する。

　　SOUND[ID=BGM FILE="ABC.ogg" FADE=1500 LOOP=3 PLAY=1]


[3]「ABC.wav」ファイルをサウンドレイヤ「BGM」にロードし、曲の３．２秒目から再生開始する。

　　SOUND[ID=BGM FILE="ABC.wav" START=3200 PLAY=1]


[4]「ABC.ogg」ファイルをサウンドレイヤ「BGM」にロードし、曲を３．２秒目から再生開始する。
　　ループして再び再生される時は、８秒目から再生される。

　　SOUND[ID=BGM FILE="ABC.ogg" START=3200 RESTART=8000 PLAY=1]


[5]「ABC.ogg」ファイルをサウンドレイヤ「BGM」にロードし、曲を１０秒後に再生開始させる。（遅延再生）

　　SOUND[ID=BGM FILE="ABC.ogg" START=-10000 PLAY=1]


[6]「ABC.wav」ファイルをサウンドレイヤ「BGM」にロードし、パンを右に設定して再生する。

　　SOUND[ID=BGM FILE="ABC.wav" PAN=512 PLAY=1]


[7]「ABC.ogg」ファイルをサウンドレイヤ「BGM」にロードし、ピッチを１．５倍にして再生する。

　　SOUND[ID=BGM FILE="ABC.ogg" PITCH=512 PLAY=1]


[8]「1.ogg」をレイヤ「BGM」へ、「2.ogg」をレイヤ「BGM2」にロードし、同時再生をおこなう。

　　SOUND[ID=BGM  FILE="1.ogg"]
　　SOUND[ID=BGM2 FILE="2.ogg"]

　　SOUND[ID=BGM  PLAY=1]
　　SOUND[ID=BGM2 PLAY=1]


[9]サウンドレイヤ「BGM」を無効にする。

　※無効にすると、ボリューム設定に関係なく、一切音が聞こえなくなります。（ミュート状態）

　　SOUND[ID=BGM E=0]


[10]サウンドレイヤ「BGM」を一時停止する。

　　SOUND[ID=BGM PAUSE=1]


[11]サウンドレイヤ「BGM」を再開する(一時停止状態をOFFにする)。

　　SOUND[ID=BGM PAUSE=0]


[12]「ABC.ogg」ファイルをサウンドレイヤ「BGM」にロードし、ボリューム値256で再生。
　　その後、１秒かけてボリューム値128の状態へフェードする。

　　SOUND[ID=BGM FILE="ABC.ogg" VOLUME=256 PLAY=1]
　　　・
　　　・
　　　・
　　SOUND[ID=BGM FADE=1000 VOLUME=128 PLAY=1]
