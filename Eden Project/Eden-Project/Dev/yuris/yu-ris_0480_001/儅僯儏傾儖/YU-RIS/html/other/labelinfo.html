■ラベル情報取得命令 LABELINFO[ ]


ラベルに関する情報を取得します。


キーワード



# = 文字列ラベル名を設定


LET = 変数名情報値を格納する変数を指定


EXIST = 値(0/1)指定したラベルの存在を調べる場合は１を指定






[1] 「SCENE01_1」というラベルがスクリプトに存在しているか否か調べる。

　　LABELINFO[#="SCENE01_1" EXIST=1 LET=@RET]
