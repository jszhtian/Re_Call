■FLASH再生命令 FLASH[ ]


FLASHファイル(swfファイル)を再生します。
FLASHファイル再生モードになると、他のどのレイヤより最前面になります。

※FLASH の再生には Internet Explorer 用の Flash Player が
インストールされている必要があります。

※swf ファイルをパックファイルに入れることは出来ません。ご了承ください。


キーワード



FILE = ファイル名FLASHファイル名を指定します


X = 値表示Ｘ座標位置を指定します


Y = 値表示Ｙ座標位置を指定します


SX = 値サイズＸを指定します


SY = 値サイズＹを指定します


PLAY = 値(0/1)１を指定すると再生します






[1]「ABC.SWF」ファイルを読み込み、ウィンドウいっぱいに広げて再生する。
  ※SX, SY キーワードを指定しなかった場合は、ウィンドウと同じサイズになります。

　　FLASH[FILE="ABC.SWF" PLAY=1]


[2]「ABC.SWF」ファイルを読み込み、左上から(50,80)の位置に、幅320、縦240のサイズで再生する。

　　FLASH[FILE="ABC.SWF" X=50 Y=80 SX=320 SY=240 PLAY=1]
