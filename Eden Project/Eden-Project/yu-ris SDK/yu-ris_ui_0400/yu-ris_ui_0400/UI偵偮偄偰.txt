■説明----------------------------------------------------------------------

YU-RIS の各種UI（ユーザーインターフェース）画像です。
Photoshop 用 psd データになります。(Photoshop7.0以降で開けます)
デフォルトで用意されているUIをちょっといじって利用したい場合などに
ご使用いただけます。
また、新規に１からUIを作成する場合でも、何かしらの参考になればと思います。

配置等、実際のサンプルとの位置と若干違っている可能性があります。
実際に組み込んだ後に微調整したりする場合があるためです。
それに関してはご了承いただければと思います。

0.400 現時点では 800x600 用のサイズのみの配布ですが、
640x480 用サイズ等も配布予定です。
