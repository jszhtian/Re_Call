立ち絵簡易マクロでの立ち絵消去の不具合について


β4.5 の ver0.458/0.09 より以前のバージョンにおいて、
立ち絵簡易マクロを使用して複数立ち絵を同時に消去しようとすると、
同時に消えず順番に消えるという不具合がありました。
サンプルとしては以下の通りです。

----------------------------------------------------------------------

\優理(yuu_1a01, 500)
\枝理(eri_1a01, 500)

表示。

\優理(, 500)
\枝理(, 500)

消去。

----------------------------------------------------------------------

これは ver0.458/0.09 で修正済みですが、
もしすでに途中までスクリプトを組んでいる方で、
この仕様を使って立ち絵を順番に消す演出を組んでいた場合、
アップデートすると演出が変わってしまいます。
そこでひとまずの対策として、UserDefine の「立ち絵定義.txt」内で、
この不具合をなくすか、そのままにするかを設定できる項目を追加しました。

「立ち絵簡易マクロを使っている方」で、
「立ち絵消去演出が変化してしまう可能性を避けたい方」のみ、
UserDefine の「立ち絵定義.txt」に、以下を追加してください。

----------------------------------------------------------------------

	//---------------------------------------------------------------------
	// 立ち絵簡易マクロを使用して複数立ち絵を消去しようとした際に、
	//   立ち絵が同時に消えない不具合をなくすか否か (1=なくす/0=なくさない)
	//
	// ※すでに以前のバージョンでスクリプトを途中まで組んでいる方は、
	// この設定により立ち絵の演出が変化してしまう可能性があります。
	//---------------------------------------------------------------------
	@T(28,91)=0

----------------------------------------------------------------------
