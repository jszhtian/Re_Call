■FLASH情報取得命令 FLASHINFO[ ]


読み込んだFLASHファイル(swfファイル)の情報を取得します。


キーワード



PLAY = 値(0/1)現在再生中(=1)か否(=0)かを調べる場合は１を指定


X = 値(0/1)表示Ｘ座標位置を取得する場合は１を指定します


Y = 値(0/1)表示Ｙ座標位置を取得する場合は１を指定します


SX = 値(0/1)サイズＸを取得する場合は１を指定します


SY = 値(0/1)サイズＹを取得する場合は１を指定します


FRAME = 値(0/1)再生中のFLASHフレーム値を取得する場合は１を指定します


TOTALFRAME = 値(0/1)読み込んだFLASHファイルのフレーム数を取得する場合は１を指定します


VAR =値(0/1)FLASH 内の SETSTR で指定した名前の変数の内容を
取得する場合は１を指定します。


SETSTR = 文字列取得する変数の名前を指定


LET = 変数名値や文字列を格納する変数を指定






[1]FLASHのＸ座標の位置を取得する。

　　FLASHINFO[X=1 LET=@POSX]


[2]現在再生中のFLASHのフレーム値を取得する。

　　FLASHINFO[FRAME=1 LET=@F]


[3]現在再生中か否かを取得する。

　　FLASHINFO[PLAY=1 LET=@FLAG]


[4]変数abc (_root.abc) の値を取得する。

　　FLASHINFO[VAR=1 SETSTR="abc" LET=@V]


[5]変数name (_root.name) の文字列を取得する。

　　FLASHINFO[VAR=1 SETSTR="name" LET=$S]
