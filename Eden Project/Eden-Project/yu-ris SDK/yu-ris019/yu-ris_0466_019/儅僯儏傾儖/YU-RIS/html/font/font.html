■フォント設定命令 FONT[ ]


テキストフォントの属性を設定します。


キーワード



NAME = フォント名文字フォント設定(フォント名指定)


NO = 値文字フォント設定(フォントリスト番号指定)


COLOR = 値(0〜0xffffff)文字色を指定


COLOR2 = 値(0〜0xffffff)文字色を指定(グラデーション用)


SX = 値文字の大きさ(横幅)を設定


SY = 値文字の大きさ(縦幅)を設定


BOLD = 値文字の太字の有無(1/0)を設定


ANTIALIASING = 値(0/1)文字のアンチエイリアス処理の有無


DRAWMODE = 値(0/1)文字を描画する際の合成方法を指定
0=透明部分も含めて新しい文字で上書き(若干高速)※デフォルト
1=合成して上書き(若干低速)






[1] テキストのフォントを、「ＭＳ 明朝」のフォントに設定する。

　　FONT[NAME="ＭＳ 明朝"]


[2] テキストのフォントを、インストールされているフォントリストの３番目のフォントに設定する。

　　FONT[NO=3]


[3] テキストサイズを横幅22、縦幅24に設定する。

　　FONT[SX=22 SY=24]


[4] テキストを太字に設定する。

　　FONT[BOLD=1]


[5] テキストの色を黄色(0xffff00)に設定する。

　　FONT[COLOR=0xffff00]


[6] テキストの色を黄色(0xffff00)から青(0x0000ff)へのグラデーション色に設定する。

　　FONT[COLOR=0xffff00 COLOR2=0x0000ff]
