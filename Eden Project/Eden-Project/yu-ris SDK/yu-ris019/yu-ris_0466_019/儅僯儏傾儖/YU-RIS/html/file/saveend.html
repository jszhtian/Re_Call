■変数保存命令 SAVEEND[ ]


SAVE 命令で予約保存されているデータを初期化（クリア）します。
※データの予約保存に関しては SAVE[ ] 命令の例[6]を参照してください。


例：　　SAVEEND[ ]


この命令を使わなくとも、SAVE[FILE="〜"] でファイルに保存すれば
自動的に初期化されますので、通常はこの命令は使う必要はありません。

とりあえずデータをメモリに格納しておいて、セーブするか否かは後で決定するような、
特殊な処理をさせたい場合などに使用します。
