■タスク終了命令 TASKEND[ ]


タスクを終了します。
タスクＩＤを指定しなかった場合、この命令を実行した自身のタスクが対象になります。


キーワード



ID = 文字列終了するタスクレイヤＩＤを指定





例：　　TASKEND[ID="SUB"]


タスクを終了する命令です。
