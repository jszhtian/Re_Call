■TOP＞命令一覧＞メニュー系命令＞MENUINFO［］
　MENU［］ 　MENUACT［］ 　MENUINFO［］ 　MENUEND［］ 
■メニューレイヤ情報取得命令 MENUINFO[ ]


メニューレイヤ(またはレイヤセット)に関する各種情報を取得します。


キーワード



ID = 文字列レイヤＩＤを指定


LET = 変数名情報値を格納する変数を指定


EXIST = 値(0/1)レイヤの存在を調べる場合は１を指定


NAME = 値(0/1)レイヤの項目名を調べる場合は１を指定


E = 値(0/1)項目の存在を調べる場合は１を指定


A = 値(0/1)項目が灰色表示か否かを調べる場合は１を指定


Z = 値レイヤの優先順位を調べる場合は１を指定


CHECK = 値(0/1)チェックマークの有無を調べる場合は１を指定


CODE = 値(0/1)レイヤに設定されている検出用値を調べる場合は１を指定






[1] メニューレイヤ「ITEM_01」が存在するか否か調べる。

　　MENUINFO[ID=ITEM_01 LET=@CHECK EXIST=1]


[2] メニューレイヤ「ITEM_01」の有効無効状態を調べる。

　　MENUINFO[ID=ITEM_01 LET=@CHECK E=1]


[3] メニューレイヤ「ITEM_01」の項目名を調べる。

　　MENUINFO[ID=ITEM_01 LET=$RET NAME=1]


[4] メニューレイヤセット「ITEM_SET」が存在するか否か調べる。

　　MENUINFO[ID=ITEM_SET/ LET=@CHECK EXIST=1]
