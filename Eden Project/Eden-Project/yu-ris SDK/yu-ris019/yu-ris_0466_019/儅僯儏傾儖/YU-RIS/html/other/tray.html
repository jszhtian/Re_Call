■トレイアイコン制御命令 TRAY[ ]


Windows のトレイアイコンの表示・非表示を制御します。
表示されるアイコンはEXEアイコンが縮小されたもの(16x16ピクセル)になります。


キーワード



NAME = 文字列トレイアイコンの上にマウスを乗せたときに
ポップアップで表示する名前を設定


ID = 文字列トレイアイコンとして表示させるＣＧレイヤＩＤを指定


MENUID = 文字列トレイアイコンの上で右クリックしたときに
表示させるメニューレイヤセットを指定


E = 値(0/1)トレイアイコンの有効無効を指定
デフォルトは０なため、ここを１にして初めて
トレイアイコンが表示されます






[1] トレイアイコンを表示する。アイコンの上にマウスを乗せると「マイアプリ」と表示され、
　　アイコンの上で右クリックすると、レイヤセット「MYMENU」内のメニューレイヤがコンテキストメニュー(ポップアップメニュー)として表示される。

　　TRAY[E=1 NAME="マイアプリ" MENUID="MYMENU/"]


[2] トレイアイコンを非表示にする。

　　TRAY[E=0]
