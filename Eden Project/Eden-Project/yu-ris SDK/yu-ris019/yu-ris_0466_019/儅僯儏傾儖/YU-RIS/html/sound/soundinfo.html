　SOUND［］ 　SOUNDINFO［］ 　SOUNDEND［］ 
■サウンドレイヤ情報取得命令 SOUNDINFO[ ]


サウンドレイヤに関する各種情報を取得します。


キーワード



ID = 文字列サウンドレイヤＩＤを指定


LET = 変数名情報値を格納する変数を指定


EXIST = 値(0/1)レイヤの存在を調べる場合は１を指定


FILE = 値(0/1)レイヤに読み込まれたファイル名を調べる場合は１を指定


E = 値(0/1)レイヤの有効無効を調べる場合は１を指定


PLAY = 値(0/1)現在再生されているか否かを調べる場合は１を指定


LOOP = 値(0/1)設定されたループ回数を調べる場合は１を指定


PAUSE = 値(0/1)現在一時停止されているか否かを調べる場合は１を指定


LOOPCOUNT = 値(0/1)再生が何回ループしたかを調べる場合は１を指定


START = 値(0/1)設定された開始位置を調べる場合は１を指定


RESTART = 値(0/1)設定されたループ後の開始位置を調べる場合は１を指定


END = 値(0/1)設定された終端位置／ループ位置を調べる場合は１を指定


VOLUME = 値(0/1)レイヤに設定されたボリューム値を調べる場合は１を指定


PAN = 値(0/1)レイヤに設定されたパンの設定値を調べる場合は１を指定


PITCH = 値(0/1)レイヤに設定されたピッチの設定値を調べる場合は１を指定


TIME = 値(0/1)現在の再生位置（単位：ミリ秒）を調べる場合は１を指定


TOTALTIME = 値(0/1)サウンドの長さ（単位：ミリ秒）を調べる場合は１を指定


CHANNEL = 値(0/1)サウンドデータのチャンネル数を調べる場合は１を指定


BIT = 値(0/1)サウンドデータのビット数を調べる場合は１を指定


SAMPLE = 値(0/1)サウンドデータのサンプリング数を調べる場合は１を指定






[1] サウンドレイヤ「BGM」が存在するか否か調べる。

　　SOUNDINFO[ID=BGM LET=@CHECK EXIST=1]


[2] サウンドレイヤ「SE」のボリューム値を調べる。

　　SOUNDINFO[ID=SE LET=@CHECK VOLUME=1]


[3] サウンドレイヤ「BGM2」のパンの設定値を調べる。

　　SOUNDINFO[ID=BGM2 LET=@CHECK PAN=1]


[4] サウンドレイヤ「VO」のピッチの設定値を調べる。

　　SOUNDINFO[ID=VO LET=@CHECK PITCH=1]
