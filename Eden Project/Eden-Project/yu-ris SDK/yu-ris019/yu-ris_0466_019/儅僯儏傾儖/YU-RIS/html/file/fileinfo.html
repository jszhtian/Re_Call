■ファイル情報取得命令 FILEINFO[ ]


指定した任意のファイルに関する情報を取得します。


キーワード



FILE = ファイル名ファイル名を指定します。※FOLDERと同時に指定は不可


FOLDER = フォルダ名フォルダ名を指定します。※FILEと同時に指定は不可


LET = 変数名各種情報値を格納する変数を指定


EXIST = 値(0/1)ファイル／フォルダの存在を調べる場合は１を指定


SIZE = 値(0/1)ファイルのサイズを調べる場合は１を指定


RELATION = 値(0/1)ファイルに関連付けされている実行ファイル名を調べる場合は１を指定






[1]「save/save_001.sd」というファイルが存在するか否か調べる。
　　※ver0.276 よりパックファイル内のファイルもチェックできるようになりました。

　　FILEINFO[FILE="save/save_001.sd" LET=@CHECK EXIST=1]


[2]「cg」フォルダが存在するか否か調べる。
　　※パックファイル内のフォルダの存在はまだチェックできません。

　　FILEINFO[FOLDER="cg" LET=@CHECK EXIST=1]


[3]「save/save_001.sd」というファイルのファイルサイズを取得する。
　　※パックファイル内のファイルのサイズはまだチェックできません。

　　FILEINFO[FILE="save/save_001.sd" LET=@FSIZE SIZE=1]


[4]「script/sample.yst」ファイルに関連づけされている実行ファイル名を取得する。
　　（例えば「メモ帳」が関連づけられていれば、notepad.exe のフルパスが取得できます）

　　FILEINFO[FILE="script/sample.yst" LET=$FNAME RELATION=1]
