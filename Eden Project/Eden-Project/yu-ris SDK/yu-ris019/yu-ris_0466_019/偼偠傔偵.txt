■最初に--------------------------------------------------------------------

　YU-RIS(ユーリス)は、簡単にゲーム等を制作するためのエンジンです。
特にビジュアルノベルやアドベンチャーゲームの制作に適しています。
Ｃ言語等のプログラミング言語を習得することなく、比較的容易で直感的な
スクリプトの記述で、制作者好みのゲームを作っていくことが出来ます。

　YU-RISシステム自体はフリーウェアであり、このシステムを使って作られたソフト
ウェアは、同人ベースにおいて許諾や報告の必要なく、自由に頒布、配布する事が
できます。また商業ベースでも条件付きで自由にお使いいただけます。
詳細は別途リファレンスマニュアルをご覧ください。


■留意事項------------------------------------------------------------------

　現段階ではシステムはβバージョンという位置付けにあり、正式版までに
仕様変更が発生する場合があります。どうかご理解ご協力お願いいたします。


■対応機種------------------------------------------------------------------

OS       : 日本語版 Windows2000/XP/2003/Vista/7/8
CPU      : PentiumIII-500MHz 以上
解像度   : 640x480 以上
カラー   : ハイカラー(16bit-Color) 以上
メモリ   : 128MB 以上（256MB 以上推奨）
サウンド : DirectSound対応サウンドカード
DirectX  : 9.0以上

CPU      : PentiumIII-500MHz 以上
解像度   : 640x480 以上
カラー   : ハイカラー(16bit-Color) 以上
メモリ   : 128MB 以上（256MB 以上推奨）
サウンド : DirectSound対応サウンドカード
DirectX  : 5.0以上


■YU-RISシステムインストール方法--------------------------------------------

基本的にインストールは行いません。
LZHファイル または 自己解凍ファイル を適当なフォルダに解凍して下さい。


■YU-RISシステムアンインストール方法----------------------------------------

解凍したフォルダをそのまま削除してください。
レジストリ関係は一切操作していません。


■YU-RIS使用方法------------------------------------------------------------

同梱のマニュアル(html形式)のほうをご参照ください。


■ファイル構成--------------------------------------------------------------

/システム/yu-ris.exe          :起動ファイル
         /エンジン設定.exe    :エンジン動作設定ファイル
         /yscfg.dat           :エンジン動作設定データ
         /system/forWin98/    :Win98以降版用EXEファイル
                /YSCom/       :コンパイルツール
                /YSConv/      :画像形式変換ツール
                /YSEnv/       :環境情報取得ツール
                /YSIcon/      :EXEアイコン書換ツール
                /YSOgg/       :OGGファイル変換ツール
                /YSOne/       :1ファイルEXE作成ツール
                /YSPac/       :ファイルパックツール
                /YSScr/       :スクリーンセーバー作成ツール
                /YSSelf/      :自己解凍EXE作成ツール
                /YSYmv/       :YMVムービー作成ツール
                /ysconfig.txt :システム設定ファイル
         /data/               :ERISサンプルプロジェクト

/マニュアル/                  :β4暫定マニュアル

/はじめに.txt                 :簡易説明テキスト(このテキスト)
