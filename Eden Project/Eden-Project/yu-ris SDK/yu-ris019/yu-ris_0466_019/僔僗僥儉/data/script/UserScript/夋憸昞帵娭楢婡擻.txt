#=フェイスウィンドウサンプル


\BGM(BGM03)


フェイスウィンドウサンプルです。


\FACE(yuu1101)

表示しました。（文字見づらくてすみません；） //サンプルが、フェイスウィンドウ用のデザインでないので、すみません；

今回のβ４から、フェイスウィンドウのＺ値を自由に設定できるようになりました。（ただし固定です）

\FACE(eri1001)

また、フェイスウインドウの後ろに背景画像を表示することも可能になりました。

背景画像の指定は、メイン画面設計で、「TIP.FACE.BACK」というボタン名で画像を読み込んでおけばＯＫです。\R必要なければ定義を外せばＯＫです。

\FACE(yuu2304)

背景画像もフェイスウィンドウに合わせて自動的に表示・非表示してくれます。

フェイスウィンドウの読み込み方法は以上です。


\FOUT(1000, 0) //幕を下ろす

\FIN(0) //幕を上げる

\GO(画像表示関連機能)

#=日付表示サンプル


\BGM(BGM03)

\BG(BG_040)

日付表示サンプルです。


\DATE(7,9,金)

\DATE.SHOW

日付表示しました。

\DATE(7,10,土)

日付変更しました。

\DATE.HIDE

日付を一時的に非表示にしました。

\DATE(7,11,日)

その状態で日付を変更しました。（ただし見えません）

\DATE.SHOW

日付を表示しました。


\FOUT(1000, 0) //幕を下ろす

\DATE.HIDE //日付消す

\FIN(0) //幕を上げる

\GO(画像表示関連機能)

#=白ヌキサンプル


\BGM(BGM03)


白ヌキサンプルです。


//白ヌキ専用命令 \EV_W
//
//EV画像を読み込むと同時に、末尾に「_w」のついたファイル名
//(このサンプルの場合だとeri_02_a_w)も同時に読み込みに行き、
//非表示状態でロードされます。(存在しない場合はロードされません)
//セーブサムネイル用にスクリーンショットを録る際に、自動的に
//白ヌキ画像と合成されて、サムネイルが作成されます。
//
\EV_W(eri_02_a)

絵が表示されている間にセーブしてみてください。

サムネイルに白ヌキが追加されているかと思います。

\EV(eri_01_a)

背景やイベント絵を変更すると、白ヌキ画像も一緒に削除されます。

白ヌキ画像の読み込み方法は以上です。


\FOUT(1000, 0) //幕を下ろす

\FIN(0) //幕を上げる

\GO(画像表示関連機能)
