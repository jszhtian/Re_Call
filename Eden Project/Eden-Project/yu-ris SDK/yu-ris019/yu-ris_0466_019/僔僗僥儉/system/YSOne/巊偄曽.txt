■注意事項------------------------------------------------------------------

YU-RIS One File Packager は、yu-ris.exe に ypfファイルを結合するツールです。
ypf ファイルは１つだけ結合することができ、結合することにより１ファイルだけで
起動が可能になります。
スクリーンセーバーを作成する場合はこのツールではなく YSScr.exe を使ってください。

■使い方--------------------------------------------------------------------

・YSOne.exe のある所と同じフォルダに data.ypf をコピーします。
  ※data.ypf ファイルの中には必ずバイナリデータ(ysbinフォルダデータ)を
    含めてください。
  ※仕様により 200MB を超える ypf ファイルは結合できません。

・YSOne.exe のある所と同じフォルダに yu-ris.exe をコピーし、
  YSOne.exe を実行します。

・生成するEXEファイル名を指定し、「EXEを生成」ボタンを押すと、
  ypfファイルが結合されたEXEファイルが同フォルダに生成されます。
